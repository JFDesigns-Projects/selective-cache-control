Selectively disable caching on specific pages to prevent server-side cache issues (especially with WordPress nonces and forms). Force reload on browser back button. Includes admin bar badge.

== Description ==

Tired of hosting cache breaking your site forms? This plugin lets you pick pages where you want strict no-cache headers, and optionally force a reload when users hit the back button. Designed to work even on aggressive caching hosts like GoDaddy.

== Installation ==

1. Upload `selective-cache-control` folder to `/wp-content/plugins/`
2. Activate the plugin via WordPress Admin > Plugins.
3. Configure via Settings > Cache Control.
